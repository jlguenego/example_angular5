Understanding Zone.js

A zone is a new kind of object added by the library Zone.js. A zone serves as context to any kind of functions and is accessed by Zone.current.

  1. At the beginning, one global zone is created by the Zone.js library: Zone.current.
  2. A new zone can be created by child = z.fork(), if z is a zone.
  3. myZone.run(myFn) run the function myFn with the current zone set to myZone.
  4. Zones have a name and a parent.
  5. Zones have properties where to store any kind of contextual data.
  6. Zones have outstanding tasks.
